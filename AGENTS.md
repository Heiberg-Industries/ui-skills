# UI Skills

Design direction for AI-generated UI. This project contains 20 style guardrail files in `styles/`.

## When building UI

1. If the user specifies a style, read `styles/{style-name}.md` and follow its guardrails
2. If no style is specified, read `styles/_index.md` to see all available styles with mood/industry tags, then help the user choose
3. Apply the chosen style's rules for typography, color, spacing, components, layout, and motion
4. Stay consistent â€” don't mix styles unless asked
